Flexbox - Documentação

○ display: flex

A propriedade display: flex transforma o elemento em um flex contêiner e nos permite manipular todos os elementos filhos dentro desse display com novas propriedades. Por padrão, quando aplicamos display: flex para um elemento, todos os filhos ficam um do lado do outro como se estivessem sob o efeito de display: inline.

Exemplos:
Valores padrão (default), das propriedades do Flexbox
justify-content flex-start
flex-direction row
aling-items stretch
flex-wrap nowrap
flex-flow Propriedade abreviada para flex-direction e flex-wrap.
align-content stretch

○ justify-content

Esta propriedade define o alinhamento dos ítens ao longo do eixo principal. Ajuda a distribuir o espaço livre que sobrar no container tanto se todos os flex items em uma linha são inflexíveis, ou são flexíveis mas já atingiram seu tamanho máximo. Também exerce algum controle sobre o alinhamento de ítens quando eles ultrapassam o limite da linha.

Abaixo estão seus possíveis valores:

flex-start

É o valor padrão. Os elementos ficam grudados um do lado do outro à esquerda do flex container. Os ítens são alinhados junto à borda de início (start) de acordo com qual for a flex-direction do container.

Exemplo:

flex-end

Os elementos ficam grudados um do lado do outro à direita do flex container. Os ítens são alinhados junto à borda final (end) de acordo com qual for a flex-direction do container.

Exemplo:

center

Os elementos ficam grudados um do lado do outro no meio do flex container. Os ítens são centralizados na linha.

Exemplo:

space-between

O primeiro elemento fica totalmente à esquerda do flex container e o último fica totalmente à direita. O restante dos elementos ficam distribuídos com um espaçamento igual entre eles. Os ítens são distribuídos de forma igual ao longo da linha; o primeiro ítem junto à borda inicial da linha, o último junto à borda final da linha.

Exemplo:

space-around

Cada elemento fica com um espaçamento igual em volta dele mesmo. Isso quer dizer que o primeiro elemento vai ter um espaçamento maior à direita do que à esquerda porque vai somar com o espaçamento à esquerda do segundo elemento. Os ítens são distribuídos na linha com o mesmo espaçamento entre eles. Note que, visualmente, o espaço pode não ser igual, uma vez que todos os ítens tem a mesma quantidade de espaço dos dois lados: o primeiro item vai ter somente uma unidade de espaço junto à borda do container, mas duas unidades de espaço entre ele e o próximo ítem, pois o próximo ítem também tem seu próprio espaçamento que está sendo aplicado.

Exemplo:

space-evenly

Corrige o "problema" do valor acima. Os elementos terão um espaçamento igual em ambos os lados. Os ítens são distribuídos de forma que o espaçamento entre quaisquer dois itens da linha (incluindo entre os ítens e as bordas) seja igual.

○ flex-direction

Estabelece o eixo principal, definindo assim a direção em que os flex items são alinhados no flex container. O Flexbox é (com exceção de um wrapping opcional) um conceito de leiaute de uma só direção. Pense nos flex items inicialmente posicionaids ou em linhas horizontais ou em colunas verticais.

Abaixo estão seus possíveis valores:

row

É o valor padrão, os ítens são alinhados Os ítens são alinhados partindo da parte esquerda do contêiner sentido a parte direita, da esquerda para a direita.

Exemplo:

row-reverse

Os ítens são alinhados Os ítens são alinhados partindo da parte direita do contêiner sentido a parte esquerda, da direita para a esquerda.

Exemplo:

column

Os ítens são alinhados partindo da parte superior do contêiner sentido a parte inferior, de cima para baixo.

Exemplo:

column-reverse

Os ítens são alinhados partindo da parte inferior do contêiner sentido a parte superior, de baixo para cima.

○ align-items

define o comportamento padrão de como flex items são alinhados de acordo com o eixo transversal (cross axis). De certa forma, funciona de forma similar ao justify-content, porém no eixo transversal (perpendicular ao eixo principal).

Abaixo estão seus possíveis valores:

stretch

É o valor padrão, estica os ítens para preencher o container, respeitando o min-width/max-width), os elementos se "esticam" para que todos fiquem com a mesma altura.

Exemplo:

flex-start

ítens são posicionados no início do eixo transversal. A diferença entre eles é sutil e diz respeito às regras de flex-direction ou writing-mode, os elementos ficam todos alinhados com o topo do flex container.

Exemplo:

flex-end

Os elementos ficam todos alinhados com a base do flex container.

Exemplo:

center

ítens são centralizados no eixo transversal, os elementos ficam todos alinhados com o meio do flex container.

Exemplo:

baseline

ítens são alinhados de acordo com suas baselines, os elementos ficam alinhados com base do conteúdo textual de cada um deles.

Exemplo:

○ flex-wrap

Por padrão, os flex items vão todos tentar se encaixar em uma só linha. Com esta propriedade você pode modificar esse comportamento e permitir que os ítens quebrem para uma linha seguinte conforme for necessário.

Abaixo estão seus possíveis valores:

nowrap

É o valor padrão, os elementos vão ficar um do lado do outro mesmo que não exista mais espaço horizontal. Todos os flex items ficarão em uma só linha.

Exemplo:

wrap

Os elementos que não cabem mais no espaço lateral recebem uma quebra de linha, ou seja, vão para a linha debaixo. Os flex items vão quebrar em múltiplas linhas, de cima para baixo.

Exemplo:

wrap-reverse

Os elementos que não cabem mais no espaço lateral recebem uma quebra de linha acima, ou seja, vão para a linha de cima. Os flex items vão quebrar em múltiplas linhas de baixo para cima.

Exemplo:

○ flex-flow

A propriedade flex-flow é uma propriedade shorthand (uma mesma declaração inclui vários valores relacionados a mais de uma propriedade) que inclui flex-direction e flex-wrap. Determina quais serão os eixos pricipal e transversal do container.

○ align-content

Organiza as linhas dentro de um flex container quando há espaço extra no eixo transversal, similar ao modo como justify-content alinha ítens individuais dentro do eixo principal. Importante: Esta propriedade não tem efeito quando há somente uma linha de flex items no container.

Abaixo estão seus possíveis valores:

stretch

É o valor padrão, os ítens em cada linha esticam para ocupar o espaço remanescente entre elas.

Exemplo:

flex-start

ítens alinhados com o início do container, o valor flex-start se guia pela flex-direction, enquanto start se guia pela direção do writing-mode.

Exemplo:

flex-end

Ítens alinhados com o final do container, o valor flex-end se guia pela flex-direction, enquanto end se guia pela direção do writing-mode.

Exemplo:

center

Ítens centralizados no container.

Exemplo:

space-between

Ítens distribuídos igualmente; a primeira linha junto ao início do container e a última linha junto ao final do container.

Exemplo:

space-around

Ítens distribuídos igualmente com o mesmo espaçamento entre cada linha.

Exemplo:

space-evenly

Ítens distribuídos igualmente com o mesmo espaçamento entre eles.
